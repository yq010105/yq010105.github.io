1. 空格

语法 example 解释
ok&nbsp;ok  ok ok 它是按下 space 键产生的空格,叫不换行空格，全称是 No-Break Space
ok&ensp;ok  ok ok 等同于字体度的一半,叫“半角空格”，全称是 En Space
ok&emsp;ok  ok ok 1 em 在 16px 的字体中就是 16px,它叫“全角空格”，全称是 Em Space
ok&thinsp;ok  ok ok “窄空格”，全称是 Thin Space。占据的宽度比较小。它是 em 之六分之一宽
ok&zwj;ok‍ ok‍ok 它叫零宽连字，全称是 Zero Width Joiner，简称“ZWJ”，是一个不打印字符
